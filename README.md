## Building

### Dependencies

On Debian:

    apt-get install build-essential libtool automake pkg-config
    wget https://www.auto.tuwien.ac.at/~mkoegler/pth/pthsem_2.0.8.tar.gz
    tar xzf pthsem_2.0.8.tar.gz
    cd pthsem-2.0.8
    ./configure
    make
    make install

### knxd

    ./bootstrap.sh
    LD_LIBRARY_PATH=/usr/local/lib ./configure
    make && make install
